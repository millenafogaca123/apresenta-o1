Coin Runner

Projeto Coin Runner

Um jogo web desenvolvido com HTML, CSS e JavaScript.

  • Objetivo: coletar moedas e desviar de obstáculos
  • Tempo de partida: 60 segundos

Objetivo do Jogo

  • Controlar o jogador
  • Coletar moedas para ganhar pontos
  • Desviar de obstáculos
  • Jogar dentro do limite de 60s

Estrutura (HTML)

  • player – personagem
  • score-display – pontuação
  • timer-display – tempo
  • start-screen – tela inicial
  • game-over-screen – tela de fim

Aparência (CSS)

  • Área do jogo e layout
  • Player, moeda e obstáculos
  • Animações com @keyframes

Lógica (JavaScript)

  • startGame(), endGame(), restartGame()
  • spawnCoins() e collectCoin()
  • spawnObstacles() e checkCollision()
  • startGameTimer()

Fluxo do Jogo

  • Tela inicial → Jogar
  • Movimentação do jogador
  • Moedas caem → pegar = +10
  • Obstáculos caem → colisão = Game Over
  • Tempo 0 → pontuação final

Conclusão

  • HTML – Estrutura
  • CSS – Aparência
  • JavaScript – Lógica
  • Resultado: jogo funcional
